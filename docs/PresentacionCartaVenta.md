# Presentacion de la Necesidad y Carta de Venta

## Presentacion de la necesidad
En laboratorios, aulas y departamentos de audiovisuales, el control manual de equipos suele provocar retrasos, extravios y falta de evidencias cuando un equipo se devuelve con fallas. Este proyecto responde a esa necesidad mediante una solucion digital capaz de conocer en tiempo real:
- Que equipos estan disponibles.
- Quien tiene un equipo prestado.
- Cuando debe ser devuelto.
- Si fue entregado con danos.

## Propuesta de valor
El **Gestor de Prestamo de Equipos Audiovisuales** mejora la organizacion institucional porque:
- Reduce errores humanos.
- Aumenta la trazabilidad de los prestamos.
- Facilita el control de inventario.
- Permite registrar danos y dar seguimiento al mantenimiento.

## Carta de venta
Estimado responsable del area audiovisual:

Le presentamos una solucion de software disenada para optimizar la administracion de camaras, proyectores y accesorios dentro de su institucion. El sistema permite registrar cada prestamo con datos del usuario, controlar disponibilidad en tiempo real y documentar devoluciones con observaciones y danos.

A diferencia de un registro manual, esta propuesta aporta orden, rapidez y evidencia historica para la toma de decisiones. La arquitectura por capas y el uso de base de datos garantizan una base solida para futuras mejoras, como autenticacion, reportes y panel web.

Esta herramienta representa una inversion en control, transparencia y eficiencia operativa.
